{% trans "How do I create an account?" %}

{% trans "How do I become a vendor?" %}

{% trans "Can I change my account type?" %}

{% trans "Bidding" %}

{% trans "How does bidding work?" %}

{% trans "Browse products and place bids on items you're interested in. Each bid must be higher than the current highest bid. You'll be notified if you're outbid, and you can place a new bid to stay in the auction." %}

{% trans "What happens if I win an auction?" %}

{% trans "When you win an auction, you'll receive a notification. You can then proceed to payment through our secure payment system. The vendor will ship your item once payment is confirmed." %}

{% trans "Can I retract a bid?" %}

{% trans "Bids are generally final. However, you can retract a bid within 1 hour of placing it if you made an error. Contact support immediately if you need to retract a bid." %}

{% trans "How do I list a product?" %}
